# MERN-Stack-Fitness-Tracker

A fitness tracker app

## Install frontend dependencies

In the "client" folder

```sh
$ yarn install
```

## Install backend dependencies

In the root folder

```sh
$ yarn install
```

## Run app - frontend and backend

In the root folder

```sh
$ yarn dev
```

## App Features

- Add new workout log
- Edit workout log
- Delete workout log
- All data saved to MongoDB

## Technical Features

- MongoDB
- Express
- Node
- React (Hooks)
- Reach Router
- React Bootstrap
