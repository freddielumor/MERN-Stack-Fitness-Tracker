# MERN-Stack-Fitness-Tracker

A fitness tracker app

## Install Front End Dependencies & run app

In the "client" folder

```sh
$ yarn install
```

```sh
$ yarn start
```

## Install Backend Dependencies & run Express server

In the "server" folder

```sh
$ yarn install
```

```sh
$ yarn server
```

## App Features

- Add new workout log
- Edit workout log
- Delete workout log
- All data saved to MongoDB

## Technical Features

- React (Hooks)
- Reach Router
- React Bootstrap
